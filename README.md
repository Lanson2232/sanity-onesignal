# sanity-plugin-onesignal

OneSignal Push Notification support for [Sanity.io](https://sanity.io?utm_source=sanity-plugin-onesignal)

# Installation

```
npm install sanity-plugin-onesignal
```

<br>

# Usage

Use it in your schema types:

```js
// [...]
{
  fields: [
    // [...]
    {
      name: 'onesignal',
      title: 'Notification',
      type: 'onesignal',
    },
  ]
}
```
